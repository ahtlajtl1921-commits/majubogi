# 📊 구글 시트 CSV 연동 설정 가이드

## 🎯 목표
구글 시트의 "웹에 게시(CSV)" 기능을 활용하여 서버 없이(Serverless) 프론트엔드에서 실시간으로 참여자 데이터를 가져와 웹사이트에 반영합니다.

## 📋 1단계: 구글 시트 CSV 게시 설정

### 1.1 구글 시트 접속
- 마주보기전 구글 시트 접속: https://docs.google.com/spreadsheets/d/1acRHfDOYSpkQGiYDShp9I6Fi4-SCa3IttAcsxOW3_KI/edit

### 1.2 웹에 게시 설정
1. **구글 시트 상단 메뉴** → **[파일]** → **[공유]** → **[웹에 게시]** 선택
2. **[게시 콘텐츠 및 설정]** 클릭
3. **게시할 시트 선택**:
   - '전체 문서' 대신 **'설문지 응답 시트1'** 선택
4. **출력 형식 선택**:
   - '웹 페이지' 대신 **'쉼표로 구분된 값(.csv)'** 선택
5. **[게시]** 버튼 클릭
6. **생성된 URL 복사** (형식 예시):
   ```
   https://docs.google.com/spreadsheets/d/1acRHfDOYSpkQGiYDShp9I6Fi4-SCa3IttAcsxOW3_KI/pub?gid=0&single=true&output=csv
   ```

### 1.3 자동 재게시 설정
- **"콘텐츠가 변경될 때 자동으로 다시 게시"** 체크박스 활성화
- 이렇게 하면 구글 폼에 새 응답이 들어올 때마다 자동으로 CSV가 업데이트됩니다.

## 🔧 2단계: 웹사이트 코드 적용

### 2.1 현재 적용된 기술
- **GoogleSheetsCSVService**: CSV 데이터를 파싱하여 참여자 정보 추출
- **Cache Busting**: URL에 timestamp 추가로 실시간성 확보
- **자동 매핑**: 날짜 정보를 기반으로 세션별 참여자 수 계산

### 2.2 작동 방식
```
구글 폼 제출 → 구글 시트 저장 → CSV 자동 게시 → 웹사이트에서 5분마다 fetch → 실시간 반영
```

## 🚀 4단계: 실시간 업데이트 확인

- **majubogi.skywork.website** → **모임찾기** 페이지
- **34기**: 현재 1명 (이지우님) 표시 확인

### 4.2 브라우저 콘솔 로그

### 4.3 실시간 테스트
1. **구글 폼**에 새 신청 입력
2. **3-5분 후** 웹사이트에서 참여자 수 증가 확인
3. **16명 달성** 시 "마감" 버튼으로 자동 변경

## ⚡ 5단계: 성능 최적화

### 5.1 캐시 정책
- **브라우저 캐시 방지**: URL에 timestamp 추가
- **구글 서버 캐시**: 3-5분 딜레이 (구글 정책, 제어 불가)
- **로컬 캐시**: 실패 시 이전 데이터 사용

### 5.2 업데이트 주기
- **자동 새로고침**: 5분마다 (300,000ms)
- **수동 새로고침**: 사용자가 페이지 새로고침 시 즉시
- **실패 처리**: 네트워크 오류 시 캐시된 데이터 사용

## 🔍 6단계: 문제 해결

### 6.1 일반적인 문제
1. **"현재 0명"으로 계속 표시**:
   - 구글 시트 CSV 게시 설정 확인
   - 브라우저 콘솔에서 오류 메시지 확인

2. **데이터 업데이트 지연**:
   - 구글 서버 캐시로 인한 3-5분 딜레이 정상
   - "자동 재게시" 설정 확인

3. **특정 참여자 누락**:
   - 구글 시트에서 데이터 형식 확인
   - 날짜 정보 정확성 확인

### 6.2 디버깅 방법
```javascript
// 브라우저 콘솔에서 실행:
// 1. 현재 캐시된 참여자 확인
console.log(googleSheetsCSVService.getCachedParticipants());

// 2. 마지막 업데이트 시간 확인
console.log(googleSheetsCSVService.getLastFetchTime());

// 3. 수동으로 데이터 새로고침
googleSheetsCSVService.getSessionsWithParticipants().then(console.log);
```
