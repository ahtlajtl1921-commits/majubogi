## 🔧 시스템 작동 방식

### 1. 실시간 데이터 연동
- 구글 폼에 새로운 신청자가 들어오면 자동으로 구글 시트에 저장
- 웹사이트가 5분마다 구글 시트에서 최신 데이터를 가져옴
- 참여자 수가 실시간으로 업데이트됨

### 2. 자동 매칭 시스템
- 구글 시트의 "참가 희망 기수/날짜/시간" 컬럼을 분석
- "2026년 3월 6일" → 34기 세션으로 자동 매칭
- "2026년 3월 7일" → 35기 세션으로 자동 매칭

### 3. 마감 처리
- 각 세션별 참여자가 16명에 도달하면 자동으로 "마감" 상태로 변경
- 마감된 세션의 신청 버튼은 회색으로 변경되어 클릭 불가

## 🚀 API 키 설정 방법

### 1. 구글 클라우드 콘솔에서 API 키 발급
1. [Google Cloud Console](https://console.cloud.google.com/) 접속
2. 새 프로젝트 생성 또는 기존 프로젝트 선택
3. "API 및 서비스" → "라이브러리" → "Google Sheets API" 활성화
4. "사용자 인증 정보" → "사용자 인증 정보 만들기" → "API 키"
5. API 키 복사

### 2. 환경변수 설정
`.env` 파일에 다음 내용 추가:
```env
VITE_GOOGLE_SHEETS_API_KEY=your_api_key_here
VITE_SESSIONS_SHEET_ID=1acRHfDOYSpkQGiYDShp9I6Fi4-SCa3IttAcsxOW3_KI
VITE_RESPONSES_SHEET_ID=1acRHfDOYSpkQGiYDShp9I6Fi4-SCa3IttAcsxOW3_KI
```

### 3. 구글 시트 공개 설정
1. 구글 시트에서 "공유" 버튼 클릭
2. "링크가 있는 모든 사용자" → "뷰어" 권한으로 설정
3. "완료" 클릭
